# Development Files

This directory contains development, testing, and demo files for the Holdem CLI project.

## Directory Structure

### `demos/`
Demo scripts and examples showcasing new features and functionality.

### `scripts/`
Utility scripts for development workflow, cleanup, and project management.

### `testing/`
Development testing scripts, performance benchmarks, and debugging tools.

### Root Files
- `debug` - Debug output and logs from development sessions

## Usage

These files are for development purposes only and are not part of the main application package.
